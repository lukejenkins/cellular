# Telit LM960

High-speed LTE Cat 18 module supporting up to 1.2 Gbps downlink. Uses the Telit proprietary `AT#` and `AT$` command set alongside standard 3GPP `AT+` commands.

| Field | Value |
|-------|-------|
| Chipset | Qualcomm SDX20 |
| Category | LTE Cat 18 |
| Command prefixes | `AT#` (Telit proprietary), `AT$` (Telit GNSS), `AT+` (3GPP standard) |
