# Quectel BG95-M3

Multi-mode LPWA module supporting LTE Cat M1, Cat NB2, and EGPRS. Designed for IoT applications.

| Field | Value |
|-------|-------|
| Chipset | Qualcomm MDM9205 |
| Category | LPWA (NB-IoT / LTE Cat M1) |
| Command prefix | `AT+Q` (Quectel proprietary) |
