# Quectel RM502Q

5G Sub-6 GHz module supporting SA and NSA modes.

| Field | Value |
|-------|-------|
| Chipset | Qualcomm SDX55 |
| Category | 5G Sub-6 GHz |
| Command prefix | `AT+Q` (Quectel proprietary) |
